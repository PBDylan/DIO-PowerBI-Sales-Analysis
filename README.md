# 📊 Dashboard de Análise de Vendas - Desafio DIO

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte do desafio técnico da DIO (Digital Innovation One). O objetivo é fornecer uma visão clara e dinâmica sobre o desempenho de vendas, permitindo análises por produto, segmento e período temporal.

## 🛠️ Tecnologias e Habilidades
- **Power BI Desktop**: Modelagem de dados e criação de visuais.
- **Power Query**: Processo de ETL para limpeza e transformação de dados.
- **DAX (Data Analysis Expressions)**: Criação de medidas para análises comparativas.
- **UX/UI Design**: Implementação de botões de navegação e layout intuitivo.

## 🚀 Funcionalidades Principais
- **Navegação Dinâmica**: Menu interativo para alternar entre as 3 páginas do relatório.
- **Filtros Temporais**: Análise detalhada por Ano e Mês (2013-2014).
- **Segmentação de Mercado**: Visão por canais como Government, Enterprise e Small Business.
